QR Based Attendance System

A real-time attendance system where students scan unique QR codes to mark their presence instantly. It uses MySQL for accurate record management and automates report exports, reducing faculty workload. Built with React.js, Django, SQL, and Tailwind CSS for a secure and efficient attendance solution.

Project Overview

The QR Based Attendance System is a web application developed to simplify and automate attendance tracking for classrooms. Students can mark their presence in real-time by scanning unique QR codes, removing the need for manual entry and reducing administrative workload. The system efficiently supports over 60 students simultaneously and stores attendance records by subject and date in a well-structured MySQL database, ensuring 95% accuracy. It also prevents tampering and maintains data integrity, offering a secure way to track attendance.

Faculty members benefit from automated Excel report exports, which cut down reporting time by up to 50%. The application’s responsive interface, built with React.js and styled using Tailwind CSS, ensures seamless operation across devices. The Django backend and SQL database provide reliable data management and fast processing. This solution improves classroom management, increases accuracy, and enhances the overall experience for both students and teachers.
