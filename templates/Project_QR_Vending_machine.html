QR Based Vending Machine App

A cashless vending machine solution where users scan a QR code to make quick and secure payments using Razorpay. Built with Python and React.js, it reduces transaction time, eliminates payment failures, and offers a seamless experience across devices.

Project Overview

The QR Based Vending Machine App is a modern, cashless payment solution designed to make vending machines safer and more convenient for users. By scanning a QR code, users can instantly complete transactions without physical contact, improving hygiene and reducing wait times. The system integrates Razorpay’s API for secure and reliable payment processing, minimizing transaction failures and ensuring a seamless experience. Over 50 successful transactions were completed during testing, demonstrating its efficiency and effectiveness in real-world scenarios.

The responsive frontend is built using React.js, ensuring compatibility across devices and providing a smooth user interface that enhances user satisfaction by 25%. The backend, powered by Python, supports fast processing and secure data handling. This solution not only streamlines payments but also supports the growing demand for contactless services. By reducing transaction time by 40% and eliminating cash handling, the app offers an efficient, scalable, and user-friendly solution for vending machine operations.
